# Removed Days :

This Pandas project automates monthly KPI reports from recurring CSV files with daily data, handling incorrect entries on specific days to streamline analysis.

# How It Works
It filters out problematic days you specify, lets you choose the report's start date, and provides the cleaned dataset plus a separate DataFrame of your working days for easy KPI calculations.

# Key Benefits🗝️
- Saves time on manual data cleaning for repetitive monthly tasks.

- Ensures accurate reports by isolating reliable data periods.

- Simple and flexible—ideal for consistent CSV formats.

- You can change the file and make it for more or fewer than seven days.
